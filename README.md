# Bob4z-a-thon

A Hackathon to Discover IBM Bob Premium Package for Z

2 Labs are offered in this hackathon:

The main lab **LabDiscoverCBSA** will allow you to discover the CBSA COBOL application. You will be able to inventory the application, discover the architecture, document and query programs or functions of the application. You can then study an evolution of the application with the corresponding implementation.

The following lab **LabDiscoverCBSAGlobal** allows you to extend the analysis to the application's frontend. This enables you to enrich the business process analysis.
